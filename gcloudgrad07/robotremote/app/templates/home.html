BotRemote - Sistema de Controle de Robôs Remotos

Interaja com robôs físicos em tempo real, de qualquer lugar.

Como Funciona?

Este sistema permite a usuários cadastrados comandar robôs físicos através de uma interface web intuitiva. Você pode registrar, gerenciar e controlar robôs com facilidade e segurança.

  • Registro e gerenciamento de contas de usuário.
  • Registro e configuração de robôs.
  • Envio de comandos em tempo real para qualquer robô registrado no sistema.

Tecnologias Envolvidas

O sistema utiliza tecnologias de ponta para garantir desempenho e segurança:

  • Python e Flask para o backend, garantindo uma plataforma robusta e escalável.
  • Bootstrap para um design responsivo e moderno.
  • PostgreSQL para armazenamento de dados seguro e eficiente.
  • Apache Kafka para o manejo de eventos em tempo real entre os usuários e os robôs.
